Chainlink Data Feeds provide multiple feed types for different onchain use cases.

| Feed Type                                                   | Purpose                                                                                                      |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| [Price Feeds](/data-feeds/price-feeds)                      | Decentralized, tamper-resistant asset price data used for a wide range of DeFi applications.                 |
| [SmartData](/data-feeds/smartdata)                          | Multiple-Variable Response (MVR) feeds offering structured multi-field responses beyond single price points. |
| [Smart Value Recapture (SVR) Feeds](/data-feeds/svr-feeds)  | Feeds designed to enable protocols to recapture MEV or other value leakage.                                  |
| [Rate and Volatility Feeds](/data-feeds/rates-feeds)        | Interest rate, implied/realized volatility, and related market metrics for advanced financial logic.         |
| [L2 Sequencer Uptime Feeds](/data-feeds/l2-sequencer-feeds) | Status indicators for L2 sequencer availability, enabling safer oracle-dependent logic during outages.       |
