This page provides structured datasets for Chainlink Data Streams.

These datasets contain stream IDs, associated metadata, and network information in a format designed for direct retrieval and processing.

When retrieving stream IDs programmatically or systematically, use the datasets below instead of parsing documentation pages.

## Cryptocurrency Streams

/data-streams/stream-ids/crypto.json

## Real World Asset (RWA) Streams

/data-streams/stream-ids/rwa.json

## SmartData Streams

/data-streams/stream-ids/smartdata.json

## Exchange Rate Streams

/data-streams/stream-ids/exchangeRate.json

## Tokenized Asset Streams

/data-streams/stream-ids/tokenizedAsset.json

## How to Retrieve Stream IDs

To retrieve a stream ID reliably:

1. Identify the correct stream category.

2. Fetch the dataset for that category:

/data-feeds/search.json?type={"{type}"}&network={"{queryString}"}

3. Filter the dataset by feed name to locate the desired stream ID.

This is the recommended retrieval path.

## Network Metadata

Each dataset includes the required network metadata (including verifier proxy addresses).

A separate networks dataset is available if additional context is required:

/data-streams/networks.json

## Notes

- Stream IDs are not network-specific
- Each dataset contains all streams for the category
- Filtering is typically performed client-side
- Do not attempt to extract stream IDs from rendered documentation tables
