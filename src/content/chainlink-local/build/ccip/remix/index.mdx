There is one comprehensive guides to help you test Chainlink CCIP in Remix IDE using Chainlink Local - without forking. Testing in forked environments is
currently supported only for Foundry and Hardhat.

## Guides

### [Using CCIP Local Simulator in Remix IDE](/chainlink-local/build/ccip/remix/local-simulator)

This guide helps you to test [Chainlink CCIP getting started guide](/ccip/getting-started) locally in Remix IDE. You will deploy a CCIP sender contract and a
CCIP receiver contract, and interact with the local simulator to send data from the sender contract to the receiver contract.
