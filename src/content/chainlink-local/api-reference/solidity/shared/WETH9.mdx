## WETH9

The [WETH9 contract](https://github.com/smartcontractkit/chainlink-local/blob/main/src/shared/WETH9.sol) for Wrapped Ether is included in Chainlink Local
to help simulate transactions that use native token payments.

```solidity
string name
```

```solidity
string symbol
```
