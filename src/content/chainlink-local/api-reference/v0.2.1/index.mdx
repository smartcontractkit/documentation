## CCIP

These contracts provide Cross-Chain Interoperability Protocol (CCIP) functionality in a local testing environment:

- [CCIPLocalSimulator](./ccip-local-simulator) - Local CCIP message routing simulator
- [CCIPLocalSimulatorFork](./ccip-local-simulator-fork) - CCIP simulator for forked networks
- [CCIPLocalSimulatorForkJS](./ccip-local-simulator-fork-js) - JavaScript utilities for CCIP simulation
- [MockEvm2EvmOffRamp](./mock-evm2evm-offramp) - Mock CCIP off-ramp contract
- [Register](./register) - CCIP network configuration registry

## Data Feeds

Contracts for simulating Chainlink Data Feeds:

- [AggregatorInterface](./aggregator-interface) - Basic price feed interface
- [AggregatorV2V3Interface](./aggregator-v2v3-interface) - Combined V2/V3 price feed interface
- [AggregatorV3Interface](./aggregator-v3-interface) - Extended price feed interface
- [MockOffchainAggregator](./mock-offchain-aggregator) - Mock implementation of off-chain aggregator
- [MockV3Aggregator](./mock-v3-aggregator) - Mock implementation of V3 aggregator

## Token Contracts

Standard token implementations for testing:

- [BurnMintERC677Helper](./burn-mint-erc677-helper) - Helper contract for ERC677 token operations
- [LinkToken](./link-token) - LINK token implementation
- [WETH9](./weth9) - Wrapped Ether implementation
