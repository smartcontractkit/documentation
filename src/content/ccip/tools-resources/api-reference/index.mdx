Chainlink Cross-Chain Interoperability Protocol (CCIP) provides secure cross-chain messaging and token transfers between blockchain networks.

- **[EVM-based Blockchains](/ccip/tools-resources/api-reference/evm/)**: Complete API reference for CCIP on Ethereum Virtual Machine (EVM) compatible blockchains.

- **[SVM-based Blockchains](/ccip/tools-resources/api-reference/svm/)**: Complete API reference for CCIP on Solana Virtual Machine (SVM) compatible blockchains.
