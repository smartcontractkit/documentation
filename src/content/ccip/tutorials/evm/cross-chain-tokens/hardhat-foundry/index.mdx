These tutorials guide you through deploying and configuring cross-chain tokens using Hardhat and Foundry development frameworks.

## Tutorials

- [Register from an EOA (Burn & Mint)](/ccip/tutorials/evm/cross-chain-tokens/hardhat-foundry/register-from-eoa-burn-mint-hardhat): Deploy and register tokens using the Burn & Mint mechanism with Hardhat or Foundry.

- [Register from an EOA (Lock & Mint)](/ccip/tutorials/evm/cross-chain-tokens/hardhat-foundry/register-from-eoa-lock-mint-hardhat): Deploy and register tokens using the Lock & Mint mechanism with Hardhat or Foundry.

- [Set Token Pool rate limits](/ccip/tutorials/evm/cross-chain-tokens/hardhat-foundry/update-rate-limiters-hardhat): Configure rate limiters for your token pools to control token flow.

- [Register from a Safe Smart Account (Burn & Mint)](/ccip/tutorials/evm/cross-chain-tokens/hardhat-foundry/register-from-safe-burn-mint-hardhat): Register tokens using a Safe smart account for enhanced security.

<Aside type="note" title="Prerequisites">
  Make sure you have Node.js and either Hardhat or Foundry installed. Familiarity with the [CCT
  standard](/ccip/concepts/cross-chain-token/overview) is recommended.
</Aside>
