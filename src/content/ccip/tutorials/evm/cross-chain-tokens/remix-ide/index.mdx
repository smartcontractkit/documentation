These tutorials show you how to deploy and configure cross-chain tokens using Remix IDE - no local development environment needed!

## Tutorials

- [Deploy and Register from an EOA](/ccip/tutorials/evm/cross-chain-tokens/remix-ide/register-from-eoa-remix): Learn how to deploy and register cross-chain tokens using only your browser and Remix IDE.
  - No development environment setup required
  - Great for quick testing and learning

<Aside type="note" title="Prerequisites">
  You only need a web browser and MetaMask. Familiarity with the [CCT
  standard](/ccip/concepts/cross-chain-token/overview) is recommended.
</Aside>
