<Aside type="note" title="Integrate Chainlink Local into your project">
  <Tabs sharedStore="package" client:visible>
    <Fragment slot="tab.1">npm</Fragment>
    <Fragment slot="tab.2">yarn</Fragment>
    <Fragment slot="tab.3">foundry</Fragment>
    <Fragment slot="tab.4">RemixIDE</Fragment>
    <Fragment slot="panel.1">

      If you use [NPM](https://www.npmjs.com/), install the [@chainlink/local NPM
      package](https://www.npmjs.com/package/@chainlink/local):

      ```shell
      npm install @chainlink/local@0.2.3
      ```

    </Fragment>
    <Fragment slot="panel.2">

      If you use [Yarn](https://yarnpkg.com/), install the [@chainlink/local NPM
      package](https://www.npmjs.com/package/@chainlink/local):

      ```shell
      yarn add @chainlink/local@0.2.3
      ```

    </Fragment>
    <Fragment slot="panel.3">

      If you use [Foundry](https://book.getfoundry.sh/):

      ```shell
      forge install smartcontractkit/chainlink-local@7d8b2f888e1f10c8841ccd9e0f4af0f5baf11dab
      ```

      and then set remappings to: `@chainlink/local/=lib/chainlink-local/` in either `remappings.txt` or `foundry.toml` file

    </Fragment>
    <Fragment slot="panel.4">

      Import the `CCIPLocalSimulator` contract from the `@chainlink/local` package:

      ```solidity
      import {CCIPLocalSimulator} from "@chainlink/local@0.2.3/src/ccip/CCIPLocalSimulator.sol";
      ```

    </Fragment>

  </Tabs>
</Aside>
